## Instructions to run

* Pre-requisites:
	-  [Go](https://golang.org/doc/install) version 1.11 or above
	-  [Seabolt](https://github.com/neo4j-drivers/seabolt)

* Installation directions
```bash
go get -u github.com/supercmmetry/lucy
```
